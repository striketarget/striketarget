## Technology Stack

The technology stack of the [striketarget.ai](https://striketarget.ai/) website is as follows.

- **Frontend**: React 18.3 + TypeScript + Vite
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Linting**: ESLint with TypeScript support

## Development Setup

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Linting
```bash
npm run lint
```
